# 📚 API Documentation

Once the backend is actively running, you can access the localized Swagger UI and OpenAPI documentation to explore our rigorous endpoints layout:

- **Interactive Swagger UI**: [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html)
- **Raw OpenAPI JSON Blueprint**: [http://localhost:8080/api-docs](http://localhost:8080/api-docs)

For all external service integrations, please rely on the auto-generated Swagger client tokens provided through the portal.
